# Vertex Black Glass — Plan et skills Claude Code

Ce pack ajoute au dépôt un plan directeur et 14 skills Claude Code pour auditer, planifier, exécuter et valider la refonte complète.

## Commandes principales

```text
/vertex-redesign-plan utilise l’image jointe comme référence et audite tout le dépôt
/vertex-redesign-orchestrator commence par la phase 0 et attends ma validation entre les phases
/vertex-redesign-foundations tout le socle
/vertex-redesign-qa branche courante
```

## Skills par espace

```text
/vertex-redesign-briefing
/vertex-redesign-markets
/vertex-redesign-opportunities
/vertex-redesign-portfolio
/vertex-redesign-analysis
/vertex-redesign-options
/vertex-redesign-performance
/vertex-redesign-intelligence
/vertex-redesign-system
/vertex-redesign-tracking
```

## Méthode recommandée

1. lancer `/vertex-redesign-plan` avec la capture de référence ;
2. valider le plan ;
3. lancer `/vertex-redesign-foundations` ;
4. exécuter une page à la fois ;
5. lancer `/vertex-redesign-qa` après chaque phase ;
6. ne jamais promouvoir sur `main` sans validation explicite.

Le style cible est noir profond + verre gris transparent. Le vert est réservé au positif, le rouge au négatif/risque et l’orange à la prudence/incertitude. Vertex reste strictement READONLY.
